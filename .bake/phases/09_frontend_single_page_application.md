# 09_frontend_single_page_application

## Objective
Serve complete dark-themed SPA dashboard from LittleFS /index.html with top-bar status, macro grid, interactive terminal, sliding drawer with accordion command list, help modal, and WebSocket auto-reconnect client.

## Done When
All UI elements from Section 8 render: device banner, status indicator (connected/disconnected), four macro buttons, terminal with adb$ prompt, drawer with Package Tools and Input Automation accordions, help modal with dismiss. WebSocket connects to /ws/console and displays output. Inject buttons populate terminal input. Split-screen UX instructions noted.
